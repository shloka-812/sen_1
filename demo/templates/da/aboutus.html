This application was created using Django. We have used web-scraping to get outbreak information from WHO and then plotted it on the map using the OpenLayers API. We also used the Google news API to provide daily disease news. We have taken disease related information from WHO and NHP. Django uses SQL Lite 3 to maintain all the User and Outbreak databases. We also incorporated a disease prediction algorithm based on 5 entered symptoms.
